## inOrderTraverse

![inOrderTraverse](./images/inOrderTraverse.PNG)

## postOrderTraverse

![postOrderTraverse](./images/postOrderTraverse.PNG)
- 后序遍历可以用来计算一个目录和它的子目录中所有文件所占空间的大小
```

```
